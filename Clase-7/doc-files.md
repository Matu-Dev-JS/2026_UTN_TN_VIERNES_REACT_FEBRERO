package.json: Archivo con informacion clave del producto como:
    nombre, version, dependencias, scripts

node_modules: Carpeta donde se instalan las dependencias del proyecto

package-lock.json: Archivo que contiene la informacion detallada las dependencias del proyecto

index.html: Es el archivo principal donde se montara el contenido del proyecto

src: Carpeta donde se encuentra el codigo fuente del proyecto

main.jsx: Archivo principal del proyecto


App.jsx: Punto de entrada del proyecto

REACT es un SPA (Single Page Application), es decir todo el contenido de la pagina sera montado sobre el index.html
